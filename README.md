# Simple tool for visualize data
**Setup**
1. Create environment
    ```
    python -m venv .venv
    ```
2. Activate environment

    Linux
    ```
    source .venv/bin/activate
    ```
    Windows
    ```
   .\.venv\Scripts\activate
    ```
3. Install libs
    ```
    pip install -r requirements.txt
    ```
4. Launch!
    ```
    python main.py
    ```
